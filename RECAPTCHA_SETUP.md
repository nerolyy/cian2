# Настройка Google reCAPTCHA

## Инструкция по настройке

### Шаг 1: Получение ключей от Google

1. Перейдите на сайт [Google reCAPTCHA Admin](https://www.google.com/recaptcha/admin)
2. Войдите в свой аккаунт Google
3. Нажмите кнопку **"+"** для создания нового сайта
4. Заполните форму:
   - **Метка**: Укажите название вашего сайта (например, "Мой сайт недвижимости")
   - **Тип reCAPTCHA**: Выберите **"reCAPTCHA v2"** → **"Я не робот"**
   - **Домены**: Укажите домен вашего сайта (например, `localhost` для локальной разработки или `example.com` для продакшена)
   - Примите условия использования
5. Нажмите **"Отправить"**

### Шаг 2: Копирование ключей

После создания сайта вы получите два ключа:
- **Site Key** (публичный ключ) - используется на клиенте
- **Secret Key** (секретный ключ) - используется на сервере

### Шаг 3: Настройка в проекте

1. Откройте файл `includes/recaptcha_config.php`
2. Замените `YOUR_SITE_KEY_HERE` на ваш **Site Key**
3. Замените `YOUR_SECRET_KEY_HERE` на ваш **Secret Key**

Пример:
```php
define('RECAPTCHA_SITE_KEY', '6LdXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX');
define('RECAPTCHA_SECRET_KEY', '6LdXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX');
```

### Шаг 4: Проверка работы

1. Откройте страницу регистрации (`register.php`)
2. Вы должны увидеть виджет "Я не робот" перед кнопкой отправки формы
3. Отметьте галочку и отправьте форму
4. Если все настроено правильно, форма должна работать

## Где используется reCAPTCHA

Капча добавлена на следующие страницы:
- **Регистрация** (`register.php`)
- **Вход** (`login.php`)
- **Добавление объявления** (`add_property.php`)

## Важные замечания

- Для локальной разработки используйте домен `localhost` при регистрации в Google reCAPTCHA
- Если вы используете другой домен в продакшене, добавьте его в список доменов в настройках reCAPTCHA
- Секретный ключ должен храниться в безопасности и не должен попадать в публичные репозитории
- Рекомендуется добавить `includes/recaptcha_config.php` в `.gitignore`, если вы используете Git

## Отключение reCAPTCHA (для тестирования)

Если вы хотите временно отключить проверку reCAPTCHA, просто не заполняйте ключи в `includes/recaptcha_config.php` (оставьте значения по умолчанию). Виджет не будет отображаться, но форма будет работать без проверки.
